Test_mate
Un program de antrenament matematic creat în C# pentru verificarea cunoștințelor de bază.

Ce face programul:
Generează sau solicită rezolvarea unor exerciții matematice.

Folosește structura if-else pentru a verifica dacă răspunsul introdus de utilizator este corect.

Afișează mesaje diferite în funcție de rezultat: "Corect" sau "Incorect, mai încearcă".

Concepte C# aplicate:
Operatori de comparare: == pentru a verifica egalitatea rezultatelor.

Logica if-else: Pentru a crea ramificații în funcție de acuratețea răspunsului.
